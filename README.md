# 📖 Pokédex - Primeira Geração

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um **Desafio de Projeto**, com o objetivo de praticar conceitos fundamentais de desenvolvimento web, consumo de APIs e manipulação do DOM utilizando JavaScript.

A aplicação consiste em uma **Pokédex interativa**, capaz de listar os Pokémon da primeira geração, exibindo informações como número, nome, tipo e imagem de cada Pokémon.

Os dados são obtidos dinamicamente através da **PokéAPI**, permitindo que a aplicação consuma informações reais e atualizadas diretamente da API.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5** — Estrutura da aplicação.
* **CSS3** — Estilização, responsividade e efeitos visuais.
* **JavaScript (ES6+)** — Consumo da API, manipulação dos dados e interação da página.

---

## 🌐 API Utilizada

O projeto utiliza a API pública:

* PokéAPI: https://pokeapi.co/

---

## 🎮 Funcionalidades

* Listagem dos Pokémon da primeira geração.
* Exibição do número da Pokédex formatado.
* Exibição do nome do Pokémon.
* Exibição dos tipos de cada Pokémon.
* Exibição detalha de cada Pokémon ao clicar no Pokémon.
* Filtro de buscas dos Pokémon pelo nome ou numero
* Imagens obtidas diretamente da PokéAPI.
* Paginação para carregamento de novos Pokémon.
* Layout responsivo para dispositivos móveis e desktops.
* Cards coloridos de acordo com o tipo principal do Pokémon.
* Efeitos visuais e animações nos cards.

## ⚙️ Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/ChronosShelby/pokedex-first-gen.git
```

2. Acesse a pasta do projeto:

```bash
cd pokedex
```

3. Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como o **Live Server** no Visual Studio Code.
